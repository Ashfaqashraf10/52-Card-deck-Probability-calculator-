# # 🃏 52 Card Deck Probability Calculator

## 📌 Project Overview

52 Card Deck Probability Calculator is a beginner-friendly Python project based on probability concepts and card mathematics. The project calculates the chances of drawing specific cards from a standard 52-card deck such as drawing an Ace, Heart, King, or other combinations.

The project demonstrates practical implementation of probability formulas, favorable outcomes, sample space calculations, conditional probability, and random card simulation using Python.

This project also helps in understanding how probabilities change after each card draw and how mathematical probability works in real-world card games.

---

## ✨ Features

- Standard 52-card deck generation
- Probability of drawing an Ace
- Probability of drawing a Heart
- Conditional probability calculations
- Random card drawing simulation
- Favorable outcome calculation
- Sample space analysis
- Card removal probability updates
- Beginner-friendly probability implementation
- Organized GitHub project structure
